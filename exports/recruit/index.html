Why Mighty-Link

静かな道具で、判断の品質を 1 段上げる。

派手な自動化ではなく、判断者の思考を支える道具を作っています。AI は判断を奪わず、視点を増やす。 この姿勢を 4 つの軸に分解して、設計と評価に貫いています。

01 / MISSION

「合う・合わない」を、感覚から数字へ。

SES と人事の現場で長年放置されてきた曖昧さを、4 軸スコアと根拠付きで返す。

02 / TECH

quota 枯渇でも止まらない。

Gemini live と決定的 fallback を同一スキーマで両立。信頼性を運用前提で設計。

03 / TEAM

静かに、確かに進める。

会議より docs、雰囲気より証跡。決まったことは Sheets と Calendar に即反映。

04 / GROWTH

3 週間で、効く設計を。

配属直後 3 週の伸びしろを week 1/2/3 ロードマップに落とす — 自分の成長にも適用。

What we build

Mighty Skill-Bridge — 4 軸の判断インフラ。

主力プロダクトは AI フィット診断 SaaS。経歴と案件を入力すると、Skill / Culture / Growth / Performing の 4 軸でスコア化し、推奨質問と 3 週間ロードマップまで一度に返します。

preview を、自分で触れます。

公開デモから即試せます。サンプル経歴 + サンプル案件で、約 3 秒で結果が返る挙動を体験してください。

  • FastAPI + vanilla JS の薄いスタック (前処理は徹底的に削る方針)
  • Gemini multimodal で PDF / 画像も直接受ける構成
  • decision-traceable: 全推論に audit log + 4 軸内訳が残る
  • fallback path は本物の本番に置ける品質で書く

Hiring process

4 ステップ、最短 2 週間。

カジュアル面談からスタート、技術選考は実際の codebase を見ながら対話で進めます。 書類だけで判断しません。

カジュアル面談

30 分、Mighty の方針・あなたの興味を相互理解。質問だけでも OK。

技術 / プロダクト対話

60 分、公開リポジトリの設計・判断軸を題材に対話。コーディング試験はなし。

1 日 trial

有給 1 日、現状の課題に小さく PR or docs PR で参加してもらいます。

オファー / 入社

条件提示 → 入社日調整。配属直後 3 週間は roadmap 付きでサポート。

FAQ

よくある質問。

面談で頻出する質問のうち、書ける範囲で先にお返しします。残りは面談で。

リモート可ですか?

週 2-3 リモート可。判断のスピードを保つ目的でオフィス日も大事にしています。

副業・複業は可能ですか?

本業の品質と顧客情報の取扱いに支障がない範囲で OK。事前共有 + 業務契約の整備が前提です。

給与は応相談ですか?

Roles 欄の数字はレンジの中央値。経験・実績・市場感に応じて個別に対話します。

未経験/異業種からでも応募できますか?

「判断インフラ」というテーマへの共鳴と、自分で docs を残せる思考があれば歓迎します。

選考にかかる時間は?

カジュアル面談 → オファーまで最短 2 週間。trial の有給日程を考慮して進めます。

面接でコーディング試験はありますか?

ありません。公開リポジトリの設計を題材に対話、必要なら 1 日 trial で実装をご一緒します。
